With the new Angular 8 app, you can now build while having the app load "automagically" as they say. Angular uses Typescript language which provides class-based object-oriented programming languages and support features of server site programming languag while using different types of syntax. An Angular application is buillt out of components which forms what is called a "tree structure" with parent and child components in which you can route them to one another. A component is an independent block of an extensive system that communicates with the other building blocks of the systems using inputs and outputs. These inputs and outputs have an associated view, data, and behavior and has parent and child components that allows the components to have re-usability, secure testing, maintenance, and separation of issues. Creating new components and routing them through the "app-routing.module.ts" allows for multiple components within the web page.
